## Quckstart

#### With npm:

```
npm i speengbab
```

```
yarn add speengbab
```

```javascript
const mocking = speeng("Ghengis Khan had some pretty good ideas!") // GhEngiS KhAn HaD SomE PrETTy GoOd iDeaS!

```
